# Contributing to Unhired.fun

Thanks for helping make unemployment slightly more quantifiable.

## Getting started

1. Fork the repository.
2. Create a branch for your change.
3. Run `npm install` and `npm run dev`.
4. Before opening a pull request, run:

```bash
npm run lint
npm run build
```

Keep changes focused and preserve the project's tone: the jokes should punch
up at broken hiring systems, never down at job seekers.

Bug fixes, mobile improvements, accessibility work, translations, new badges,
and scientifically questionable scoring ideas are all welcome.

## Pull requests

Explain what changed and include screenshots for visual changes. Small pull
requests are easier to review.

If your pull request sits unanswered for a few days, please don't call it
ghosting. We prefer **asynchronous contributor evaluation**.
